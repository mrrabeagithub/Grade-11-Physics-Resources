⚛️ Grade 11 Physics Resources

Welcome to the official repository for Grade 11 Physics. This library contains all the resources, simulations, and smart-sheets you need to master the curriculum, bridging the gap between theoretical mechanics and real-world applications.

📖 How to Use This Library

Navigate through the chapter folders below. Inside each chapter, you will find a curated collection of materials divided by type:

Summaries & Study Guides: Core concepts, derivations, and formulas.

Infographics: Visual intuition maps for complex physical laws.

Worksheets: Practice problem sets.

SmartSheets: Interactive HTML documents for dynamic problem solving.

Simulations: Live physics sandboxes featuring numerically stable (RK4/Semi-Implicit Euler) engines. All simulations include our standard Triple-View: Time Domain graphs, Phase Space plots, and Live Energy Bar charts.

Media: Video lectures and curated YouTube playlists.
